Events

Event Handling

System resources and time define events. All events can be handled using the every keyword.

every accelerometer.any_motion { }
every 60 seconds { }

Events can be handled multiple times. Event handlers will be executed in the order they appear in the source code.

var foo = 0;
every accelerometer.any_motion { 
    foo = 1; 
}
every accelerometer.any_motion { 
    foo = 2; 
}
